# Memex Alpha Updates

This directory is reserved for the Memex alpha Sparkle appcast:

`https://5x2ktjxtqv-ops.github.io/memex-updates/alpha/appcast.xml`

Current status: no alpha release has been published from this repository.

`appcast.xml.example` is a non-publishing example only. Real `appcast.xml`
updates and files under `releases/` must be added only by an explicit approved
release process.

Never commit Sparkle private keys or binary release artifacts to this
repository.
